# app

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Pushing to Firebase environment
Requires permission to the firebase project configured under `.firebaserc`.
* ```firebase use dev```
Or
* ```firebase use prod```
Then
* ```firebase deploy```
